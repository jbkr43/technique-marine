## Design Context

This project uses the Impeccable design system. Key reference files:

- **PRODUCT.md** — Strategic context: register (brand), users, brand personality, anti-references, design principles
- **DESIGN.md** — Visual design system: colors (Channel Deep, Brass, Dock Paper, Weathered Tin), typography (Alkaline + system-ui), component specs, elevation philosophy
- **`.impeccable/design.json`** — Machine-readable tokens with tonal ramps, component HTML/CSS snippets

Start here for any design-related work. Always read PRODUCT.md and DESIGN.md before generating new pages or components.
